# JConsoleAudioPlayer

### About: JConsoleAudioPlayer is a simple audio player for Java <br>
### It is support the following audio format: <br>
### - Wave (24000 HZ - 16 bit) <br>

### Controllers: <br>

| Controller | Function             |
|------------|----------------------|
| +          | Add to playlist      |
| -          | Remove from playlist |
| p          | Play                 |
| s          | Stop                 |
| r          | Rest                 |
 | n          | Next                 |
 | pr         | Previous             |
 | l          | Loop                 |
 | lp         | Loop Playlist        |
| sh         | Shuffle playlist     |
| m          | Mute                 |
| vl         | Show volume level    |
| vu         | Volume up            |
| vd         | Volume down          |
| :          | Search               |
| q          | Quit                 |
